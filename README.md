# E-commerce Sales Data Pipeline

This project is a simple data engineering project built using Python, PySpark, SQL, Pandas, and Matplotlib in Google Colab.

## Objective
The main goal of this project is to clean, process, analyze, and visualize e-commerce order data.

## Tools Used
- Python
- Pandas
- PySpark
- SQL
- Matplotlib
- Google Colab

## Dataset Columns
- order_id
- customer_id
- product_id
- category
- price
- discount
- quantity
- payment_method
- order_date
- delivery_time_days
- region
- returned
- total_amount
- shipping_cost
- profit_margin
- customer_age
- customer_gender

## Project Steps
1. Loaded the dataset into Google Colab
2. Cleaned the data using Pandas
3. Converted the data into a PySpark DataFrame
4. Performed grouping and aggregation using PySpark
5. Ran SQL queries for analysis
6. Created bar charts using Matplotlib

## Analysis Done
- Region-wise sales analysis
- Category-wise sales analysis
- Payment method analysis

## Output
The project generates cleaned data, summary results, and visualizations.
